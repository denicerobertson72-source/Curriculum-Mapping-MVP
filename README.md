# Curriculum Mapping Tool

## Overview

This repository contains planning documents for a homegrown curriculum mapping tool MVP.

The initial MVP is intentionally narrow and focused on helping programs map courses to program student learning outcomes using Introduced / Reinforced / Mastered (I/R/M) designations.

The MVP also includes a lightweight syllabus/topic placeholder so the tool can later support review of overlapping course content.

## MVP Scope

The MVP includes:

- Programs
- Courses
- Program SLOs
- Optional course SLOs
- Course-to-program SLO mapping
- I/R/M tagging
- Curriculum map exports
- Gap and coverage reporting
- Lightweight topic placeholder fields

## Out of Scope for MVP

The MVP does not initially include:

- Full assessment-plan workflows
- Assignment-level evidence collection
- Rubric-level scoring aggregation
- Student-level analytics
- Accreditation report automation
- Automated syllabus overlap analysis
