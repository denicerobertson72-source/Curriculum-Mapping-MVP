# User Roles and Workflows

## Admin
- Manage users
- Manage departments
- Manage programs
- Manage courses
- Import data
- Manage exports

## Program Lead / Chair
- Manage program SLOs
- Review curriculum maps
- Assign I/R/M values
- Export reports

## Faculty Editor
- View assigned courses
- Edit course SLOs
- Edit topic placeholder fields
- Propose or edit mappings if allowed

## Read-Only Reviewer
- View programs
- View maps
- Run exports

## Basic Workflow
1. Admin imports programs, courses, and SLOs
2. Faculty review course records
3. Program leads build curriculum maps
4. Reviewers export reports and check gaps
