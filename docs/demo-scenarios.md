# Demo Scenarios

## Scenario 1: Create a Program
- Add a new program record
- Assign department and degree type
- Save program details

## Scenario 2: Add Program SLOs
- Open Program SLO Management
- Add SLO number, short label, and full statement
- Save and reorder if needed

## Scenario 3: Import Courses
- Upload courses CSV
- Validate course records
- Confirm imported data

## Scenario 4: Build a Curriculum Map
- Open a program
- Launch Curriculum Map Matrix
- Assign I/R/M values by course and SLO
- Save mapping notes if needed

## Scenario 5: Export a Gap Report
- Run Gap & Coverage Report
- Review outcomes with missing I/R/M coverage
- Export PDF or CSV

## Scenario 6: Add Topic Placeholders
- Open Course Detail
- Add major topics, skills, assignments, and syllabus link
- Save for future overlap review
